Practica de transiciones y animaciones

Transicion con CSS

Permite cambiar suavemente una propiedad CSS de un estado a otro durante un periodo de tiempo

Propiedades de las Transiciones

  • transition-property:Define que propiedad cambiar.
  • transition-duration:Define la duracion del cambio.(s o m)
  • transition-timing-function:Controla la aceleracion o desaceleracion
    • linear → velocidad constante.
    • ease (por defecto) → comienza lenta, acelera y luego desacelera.
    • ease-in → comineza lenta y acelera.
    • ease-out → comienza rapida y desacelera.
    • ease-in → comineza lenta y acelera.
  • transition-delay:Retrasa el inicio de la transicion.

Transicion de color de fondo

La transicion de color de fondo en CSS permite cambiar suavemente el color de fondo de un elemento cuando se activa un cambio de estado, como un :hover.Esto crea un efecto visual agradable y fluido que mejora la experiencia del usuario al interactuar con la pagina.Por ejemplo, al pasar el cursor sobre un boton, su color de fondo puede cambiar de azul a rojo, atrayendo la atencion del usuario sin ser abrupto

Transicion de tamaño de fuente

La transicion de tamño de fuente facilita el cambio gradual del tamaño del texto de un elemento.Esto es util para destacar elementos interactivos o importantes el aumentar su tamaño cuando se pasa el culsor sobre ellos .Por eemplo un parrafo de texto puede aumentar su tamaño de fuente cuando se situa el cursor sobre el, destacando su relevancia o indiando que es interactivo.
